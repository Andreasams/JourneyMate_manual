# Business Information Page — Complete Bundle

**FlutterFlow Widget:** `BusinessInformationWidget`
**Route:** `BusinessInformation` (path: `businessInformation`)
**JSX File:** `information_page.jsx`
**Directory:** `05_contact_details` (⚠️ Misleading name - should be `05_business_information`)
**Status:** ✅ Production Ready

---

## Purpose

Full-screen dedicated information page showing comprehensive restaurant details including interactive map, opening hours, features/services, and payment methods. Accessed from Business Profile page.

**Primary User Task:** View detailed restaurant information and contact details in one place.

---

## User Flow

```
Business Profile Page
  │
  ├─ User taps "Se alle informationer →" link
  │
  ├─ Navigate to → BusinessInformationWidget
  │
  ├─ Page loads:
  │    ├─ Google Maps shows location
  │    ├─ Status calculated (open/closed)
  │    ├─ Filters loaded
  │    └─ Page start time recorded
  │
  ├─ User views:
  │    ├─ Business name & status (overlay on map)
  │    ├─ Business description (if available)
  │    ├─ Hours & contact (expandable) → ContactDetailWidget
  │    ├─ Features chips → Tap → FilterDescriptionSheet
  │    └─ Payment chips
  │
  ├─ User interactions:
  │    ├─ Back button → Mark engagement, navigate back
  │    ├─ Map marker tap → Mark engagement
  │    ├─ Hours section expand → Mark engagement, show contact details
  │    └─ Feature chip tap → Show filter description bottom sheet
  │
  └─ Page dispose → Track analytics (page_viewed with duration)
```

## Page Structure

- Left: Back button (iOS style arrow)

- Icon: `Icons.arrow_back_ios`

- Action: `await actions.markUserEngaged()` → `context.safePop()`

**Business Name:**
- Source: `FFAppState().mostRecentlyViewedBusiness.business_name`
- Font: 16px, 400 weight
- Alignment: Center
- Color: `primaryText`

### 2. Google Maps Section (200px height)

**Configuration:**
- Initial location: Business lat/long from `FFAppState().mostRecentlyViewedBusiness`
- Initial zoom: 12
- Marker color: Red (`GoogleMarkerColor.red`)
- Map type: Normal
- Style: Standard
- Allow interaction: true
- Allow zoom: true
- Show location: true (user's location)
- Show zoom controls: false
- Show compass: false
- Show map toolbar: false
- Show traffic: false

**Position:** Padding `12, 212, 12, 2` (overlays bottom of map)

**Business Name:**
- Source: `FFAppState().mostRecentlyViewedBusiness.business_name`

## Custom Actions Used

### 1. determineStatusAndColor()
- **Called:** On page load (`initState`)
- **Purpose:** Calculate current open/closed status and set status color
- **Parameters:**
  - Color callback function
  - Opening hours data
  - Current timestamp
  - Language code
  - Translations cache
- **Side Effects:** Sets `_model.statuscolor` via callback

### 2. getFiltersWithUpdate()
- **Called:** On page load (`initState`)
- **Purpose:** Load filter descriptions for current language
- **Parameters:** Language code
- **Returns:** Filter data
- **Stores:** In `_model.filters`

### 3. markUserEngaged()
- **Called:** On:
  - Back button tap
  - Map marker tap
  - Hours section expand/collapse
- **Purpose:** Track user engagement for analytics
- **Parameters:** None
- **Returns:** None (async void)

### 4. trackAnalyticsEvent()
- **Called:** On page dispose
- **Purpose:** Track page view event with duration
- **Parameters:**
  - Event name: `'page_viewed'`
  - Event data: `{ 'pageName': 'businessInformation', 'durationSeconds': calculated }`

---

## Custom Functions Used

### 1. latLongcombine()
- **Purpose:** Combine latitude and longitude into LatLng object
- **Parameters:** latitude (double), longitude (double)
- **Returns:** LatLng object for Google Maps

### 2. daysDayOpeningHour()
- **Purpose:** Generate dynamic status text based on current time and opening hours
- **Parameters:**
  - Current timestamp
  - Opening hours data
  - Language code
  - Translations cache
- **Returns:** String (e.g., "Åbner kl. 17:30", "Lukker kl. 22:00")
- **Examples:**
  - Currently closed, opens later today: "Åbner kl. 17:30"
  - Currently open, closes soon: "Lukker kl. 22:00"
  - Currently open, closes tomorrow: "Åbent til 01:00 i morgen"

### 3. getSessionDurationSeconds()
- **Purpose:** Calculate time spent on page
- **Parameters:** Page start timestamp
- **Returns:** Duration in seconds (int)

## Lifecycle Events

### initState

**Sequence:**
1. Create model: `_model = createModel(context, () => BusinessInformationModel())`
2. Post-frame callback:
   - Record page start time: `_model.pageStartTime = getCurrentTimestamp`
   - Calculate status: `_model.currentStatus = await actions.determineStatusAndColor(...)`
   - Load filters: `_model.filters = await actions.getFiltersWithUpdate(languageCode)`
3. Initialize expandable controller:
   - `_model.hoursAndContactExpandableController = ExpandableController(initialExpanded: false)`
   - Add listener to mark engagement on expand/collapse

### dispose

**Sequence:**
1. Fire-and-forget async block:
   - Track analytics: `await actions.trackAnalyticsEvent('page_viewed', { 'pageName': 'businessInformation', 'durationSeconds': calculated })`
2. Dispose model: `_model.dispose()`
3. Call super dispose: `super.dispose()`
